## 新 Agent 加载顺序

```
1. 读 CLAUDE.md（项目根目录 — 启动规范）
2. 读 index.md（本文件 — 导航地图）
3. 读 schema.md（宪法摘要 — 快速了解全局约束）
4. 读 00 宪法层/ 全部 7 个文件（完整规则）
5. 读 10 运行时层/ 全局规则（路由、偏好、状态）
6. 读 20 数据层/（模板、规范）
7. 去 profiles/ 加载当前角色身份
8. 去 项目空间/ 找到目标项目 → 读全部项目文件
```
