# 任务清单

> **层级**: 运行时层  
> **作用**: 跨项目统一任务数据库，聚合所有项目的任务状态  
> **维护者**: Coordinator

---

## 项目总览

```dataview
TABLE WITHOUT ID
  项目ID AS "项目ID",
  count(rows) AS "总任务数",
  sum(rows.状态 = "已完成") AS "已完成",
  sum(rows.状态 = "执行中") AS "执行中",
  sum(rows.状态 = "待执行") AS "待执行",
  sum(rows.状态 = "回退中") AS "回退中"
FROM "10 运行时层/项目空间"
WHERE 任务ID
GROUP BY 项目ID
SORT 项目ID ASC
```

## 活跃任务（执行中 + 回退中）

```dataview
TABLE WITHOUT ID
  任务ID AS "任务ID",
  标题 AS "标题",
  项目ID AS "项目",
  负责人 AS "负责人",
  状态 AS "状态",
  优先级 AS "优先级",
  回退次数 AS "回退",
  预估置信度 AS "置信度"
FROM "10 运行时层/项目空间"
WHERE 状态 = "执行中" OR 状态 = "回退中"
SORT 优先级 ASC, 任务ID ASC
```

## 待执行任务

```dataview
TABLE WITHOUT ID
  任务ID AS "任务ID",
  标题 AS "标题",
  项目ID AS "项目",
  类型 AS "类型",
  负责人 AS "负责人",
  优先级 AS "优先级",
  预估置信度 AS "置信度"
FROM "10 运行时层/项目空间"
WHERE 状态 = "待执行"
SORT 优先级 ASC, 任务ID ASC
```

## 高回退任务（回退 ≥ 2 次）

```dataview
TABLE WITHOUT ID
  任务ID AS "任务ID",
  标题 AS "标题",
  项目ID AS "项目",
  回退次数 AS "回退次数",
  状态 AS "状态"
FROM "10 运行时层/项目空间"
WHERE 回退次数 >= 2
SORT 回退次数 DESC
```

## 统计

```dataview
TABLE WITHOUT ID
  length(rows) AS "总数",
  sum(rows.状态 = "已完成") AS "已完成",
  sum(rows.状态 = "执行中") AS "执行中",
  sum(rows.状态 = "待执行") AS "待执行",
  sum(rows.状态 = "回退中") AS "回退中",
  round(100 * sum(rows.状态 = "已完成") / length(rows), 1) + "%" AS "完成率"
FROM "10 运行时层/项目空间"
WHERE 任务ID
```

## 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| 任务ID | Input | T001, T002... 格式 |
| 标题 | Input | 任务简要描述 |
| 项目ID | Input | 所属项目的 PRJ 编号 |
| 类型 | Select | research / write / build / hybrid / review |
| 负责人 | Select | Coordinator / Researcher / Writer / Builder |
| 状态 | Select | 待执行 / 执行中 / 已完成 / 回退中 / 已驳回 |

| 回退次数 | Number | 0 起步 |
| 优先级 | Select | P0(紧急) / P1(高) / P2(中) / P3(低) |
