# 信息源技术可行性报告

> **任务**: T001  
> **角色**: Researcher  
> **方法**: 逐一评估 7 个信息源的获取可行性、技术方案、置信度和风险

---

## 1. 海外独角兽

| 维度 | 评估 |
|------|------|
| **类型** | 微信公众号（投资研究向） |
| **获取方式** | 用户手动复制文章链接 → 通过 wechat-article-exporter 导出全文 |
| **技术可行性** | 仅能半自动：用户需提供链接，工具可导出 |
| **频率** | 约每周 2-3 篇 |
| **置信度** | [置信度:0.85] — 导出工具经社区验证有效，但依赖用户手动操作 |
| **风险** | 部分文章可能付费墙拦截；工具依赖微信公众平台页面结构，改版可能失效 |

**结论**: ✅ 可行，但需要用户配合提供链接

## 2. Dwarkesh Podcast（B站搬运）

| 维度 | 评估 |
|------|------|
| **类型** | B站视频（Bilibili） |
| **获取方式** | bilibili-api-python 获取视频信息 + AI 字幕 |
| **技术可行性** | 已验证：QR 码登录成功，凭证已保存 ✅ |
| **频率** | 约每周 1-2 期 |
| **置信度** | [置信度:0.90] — 字幕 API 稳定，QR 登录已验证，但 AI 字幕可能有少量识别错误 |
| **风险** | B站 API 可能改动；凭证有效期有限（可设置自动刷新） |

**结论**: ✅ 全自动可行。下一步：写字幕获取脚本

**来源标签建议**: `[外部-Bilibili/BV号]`

## 3. 张小珺Jùn｜商业访谈录（小宇宙播客）

| 维度 | 评估 |
|------|------|
| **类型** | 小宇宙播客 |
| **获取方式** | RSSHub RSS 拉取 shownotes + 元数据 |
| **技术可行性** | RSSHub 可返回节目标题、描述、shownotes、发布时间 |
| **AI 逐字稿** | 小宇宙有 AI 逐字稿功能（Plus 会员），但官方未开放 API 获取 [置信度:0.60] |
| **置信度** | 仅 shownotes: [置信度:0.55]；若有逐字稿: [置信度:0.80] |
| **限制** | 纯音频我无法处理；shownotes 内容有限，无法支撑高置信度 |

**结论**: ⚠️ 先上 RSS shownotes（标记完整度不足），遇到精读节目走 BibiGPT 或你手动补逐字稿

**来源标签建议**: `[外部-小宇宙/播客名/期号]`

## 盲区声明

- **纯音频处理**：我无法处理任何音频格式（MP3/WAV/AAC），此能力完全不覆盖
- **逐字稿验证**：小宇宙 AI 逐字稿的完整性和准确性未经验证 [置信度:0.60]
- **B站字幕时效**：AI 字幕可能存在口音/专业术语识别错误 [置信度:0.70]
- **RSSHub 稳定性**：依赖第三方服务 rsshub.app，可能因流量限制或维护而不可用
