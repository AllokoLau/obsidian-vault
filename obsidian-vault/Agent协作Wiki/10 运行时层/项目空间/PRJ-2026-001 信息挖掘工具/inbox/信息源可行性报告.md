# 信息源技术可行性报告

> **任务**: T001  
> **角色**: Researcher  
> **方法**: 逐一评估 7 个信息源的获取可行性、技术方案、置信度和风险

---

## 1. 海外独角兽

| 维度 | 评估 |
|------|------|
| **类型** | 微信公众号（投资研究向） |
| **获取方式** | 用户手动复制文章链接 → 通过 wechat-article-exporter 导出全文 |
| **技术可行性** | 仅能半自动：用户需提供链接，工具可导出 |
| **频率** | 约每周 2-3 篇 |
| **置信度** | [置信度:0.85] — 导出工具经社区验证有效，但依赖用户手动操作 |
| **风险** | 部分文章可能付费墙拦截；工具依赖微信公众平台页面结构，改版可能失效 |

**结论**: ✅ 可行，但需要用户配合提供链接

## 3. 张小珺Jùn｜商业访谈录（B站视频）

| 维度 | 评估 |
|------|------|
| **类型** | B站视频（自有账号，非搬运） |
| **获取方式** | B站 API 获取视频信息 + AI 字幕（UID: 280780745） |
| **技术可行性** | B站 API 已验证可用（搜索 + 空间 + 视频信息） |

**结论**: ✅ 全自动可行。与 Dwarkesh Podcast 同方案（B站 API + 字幕提取）

**来源标签建议**: `[外部-Bilibili/BV号]`

## 4. 晚点LatePost（微信公众号）【2026-06-24 更新：由B站改为微信公众号】

| 维度 | 评估 |
|------|------|
| **类型** | 微信公众号（深度商业报道） |
| **获取方式** | 用户提供微信公众号文章链接（mp.weixin.qq.com/s/xxx）→ wechat_link_processor.py 抓取全文 |
| **技术可行性** | 半自动：需用户提供链接；脚本已验证可抓取微信公众号全文（支持海外独角兽方案） |
| **频率** | 持续更新，AI/科技/汽车/电商领域深度独家报道，日更多篇 |
| **置信度** | 微信公众号全文可达 [置信度:0.85]（权威性权重0.85，可交叉验证的独家报道） |
| **替代渠道** | latepost.com（WebFetch受限）、163.com（受限）、jiemian.com（受限）— 均不可直接抓取 |
| **近期重磅内容** | 字节Seed架构调整 / 阿里复盘外卖大战 / 理想战略换挡 / 字节AI账本 / 赛力斯新品牌 / SpaceX IPO解密 |
| **风险** | 需要你提供链接；微信公众号可能有反爬限制 |

**结论**: ⚠️ 半自动可行，需要你提供链接。内容质量极高（独家深度报道），比B站短视频价值大得多

**来源标签建议**: `[外部-微信公众号/文章标题]`
