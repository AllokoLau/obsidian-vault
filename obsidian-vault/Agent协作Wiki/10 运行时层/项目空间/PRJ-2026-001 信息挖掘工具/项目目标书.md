## 3. 统一处理管道

```
原始内容 → raw/（按模板入库）
         → Researcher 置信度评估
            ├─ < 0.4 → 丢弃
            ├─ 0.4-0.7 → process/（等人为判断）
            └─ > 0.7 → wiki/（直接入库）
```

## 5. 约束条件

- 遵循 00 宪法层/ 所有规则

- 微信公众号内容需用户提供链接
