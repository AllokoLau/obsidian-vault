## 技能加载规则

1. Coordinator 路由任务时，根据分配表将对应技能内容注入目标 Profile 的 Prompt
2. 每个 Profile 的 `skills/README.md` 引用本表中分配给它的技能
3. 技能文件本体保存在 vault 根目录 `SKILL/`，各 Profile 不复制
4. 新增技能必须先在本表登记，说明分配对象和使用场景
