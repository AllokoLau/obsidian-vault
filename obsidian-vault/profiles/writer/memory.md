# Writer 经验

## 写作经验（跨项目可复用）

- 播客精炼格式：YAML frontmatter → 核心结论 → 章节速览(表格) → 关键要点(编号) → 精彩引述(> 引用)
- 时间戳用 `[MM:SS]` 格式
- 初稿完成后路由给 Researcher 做 fact-check
- 材料不足时向 Coordinator 申请回退，不自行脑补

## 注意

- 项目特定的写作内容不存于此 → 存到 `项目空间/{项目}/outputs/`
