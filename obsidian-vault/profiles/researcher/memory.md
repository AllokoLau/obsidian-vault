# Researcher 经验

## 调研经验（跨项目可复用）

- 优先找一手来源（原始报告、官方文档），二手来源（转述、摘要）降一档置信度
- 信息不足时扩大检索范围，不强行给出高置信度结论
- 来源冲突时增加验证来源数量，以质量更高的结论为准
- 产出放 `inbox/`，不直接写入最终位置

## 注意

- 项目特定的调研结论不存于此 → 存到 `项目空间/{项目}/inbox/`
- 临时检索笔记不存于此 → 存到 `inbox/`
