# 卖在人声鼎沸处，就是趋势结束时？

> **嘉宾**: 丁莹（康曼德资本董事长，30年证券从业经历）
> **主持人**: 齐一（雪球《厚雪长波》）
> **时长**: ~100分钟

## 核心结论

丁莹从1994年入行的"红马甲"场内交易员，到28岁管理20亿公募基金，再到创办康曼德资本，完整穿越了中国资本市场五次牛熊。他的投资哲学浓缩为12个字——**跟随趋势、学习常识、拥抱创新**，并以此构建了一套融合主观与量化、基本面与交易策略的3.0版价值投资体系。

核心观点：
1. **"卖在人声鼎沸处"就是趋势结束时**——二者是一回事，而非矛盾。趋势的起点和终点都有可识别的产业信号和估值信号。
2. **用"投资速度"公式做调仓决策**：投资速度 = 空间 ÷ 时间 × 概率，把资金从速度慢的品种切换到速度快的品种。
3. **AI是"效率助理"，不是替代者**：丁莹自建AI Agent每天处理全市场信息，但"AI用的越多就越需要实地调研"——超额收益来自物理世界的一手信息。
4. **当前处在康波周期的AI大趋势中**：A股正从流动性驱动转向盈利驱动，AI基建（算力/存储/能源）之后必是AI应用的大机会。
5. **投资体系的闭环最重要**：从点状认知到形成SAB-KMD评级体系再到AI赋能，花了近10年才完成闭环，"体系的力量重于某支股票给你的贡献"。

### 关键要点

#### 1. 投资哲学：12字总纲 + 三层迭代

丁莹的核心框架是12个字：**跟随趋势、学习常识、拥抱创新**，三者构成一个完整闭环：
- **跟随趋势** → 止损/不犯致命错误（必要条件）
- **学习常识** → 跨学科知识（自然科学/社会科学/心理学）→ 提高胜率
- **拥抱创新** → 获得超额赔率（充分条件）

他将价值投资分为三个版本：
> **1.0（格雷厄姆式）**：比净资产便宜才买
> **2.0（巴菲特式）**：合理价格买入好公司
> **3.0（搏际投资式）**：看未来影响人类的科技方向，长期持有

#### 2. SAB-KMD评级体系

这是丁莹团队花了近10年打磨的投资框架：

| 评级 | 含义 |
|------|------|
| **S** | 好赛道 + 好公司 → S5（万亿级）/ S4（千亿级）/ S3/S2/S1 |
| **A** | 好赛道 + 一般公司 |
| **B** | 一般赛道 + 好公司 |
| **C** | 不投 |
| **K** | 重仓（目标价严重低估 + 催化剂） |
| **M** | 持有（估值合理） |
| **D** | 清仓（严重高估） |

#### 3. "投资速度"公式

> **投资速度 = 空间（预期涨幅）÷ 时间（预期到达时间）× 概率（确定性）**

计算每只股票的投资速度后，把资金从速度慢的品种切到速度快的品种。这个公式是调仓换仓的核心决策工具。

#### 4. AI Agent实战案例

丁莹自建了一个个人使用的AI投资系统，每天的工作流：

| 时间 | AI任务 |
|------|--------|
| 07:30 | 股票池所有股票的边际变化表格 |
| 08:00 | 估值高低提示（高估/低估清单） |
| 08:30 | 市场动量较强的股票提示 |
| 09:00 | 输出投资建议（评级+交易策略） |

AI的作用定位：**效率助理**，而非决策替代。
- "AI做的是一致性预期的梳理"
- "超额收益来自找到一致性预期错在哪里"
- "AI用的越多就越需要调研——人要验证AI的结论，获得网上没有的真实信息"

#### 5. 对当前AI行情的判断

> **"AI的基建是修路，AI的应用是跑车。我们现在短期看的是AI的基建，但长期一定要看AI的应用。"**

三层投资逻辑：
1. **基建层**（当前主线）：算力、存储、能源管理系统 → 业绩已释放
2. **应用层**（下一阶段）：AI应用将出现大公司，空间更大
3. **赋能层**（长期）：AI改造各行各业效率，消费/制造/服务都有机会

#### 6. 风险控制的减法思维

丁莹的对冲策略独特之处：不做空自己的持仓股。
- 用股指期货控制整体敞口（3-5成对冲）
- "要减震的时候，卖别人的股票，不卖自己的股票"
- 如果自己的股票想卖，就直接清仓，不会一边持有一边做空
