## 正文

### Agent 时代的网络安全变革

**攻击侧**：网络攻击综合成本下降约 100 倍至 1000 倍。过去一个高质量零日漏洞从研究到武器化成本在 10 万到 250 万美元之间，现在普通人借助 agent 的 vibe coding 仅需几秒钟就能生成攻击代码。社会工程学攻击在 Deepfake 等 AI 加持下，个性化定制成本被抹平，2024 年下半年凭证钓鱼攻击暴增 703%。

Agent 具备推理、学习和自我复制能力，不需要休息，能 7x24 持续攻击，同时针对成百上千个目标发起并发打击。一旦 agent 成功执行某种攻击，该攻击的发生概率就是 100%——AI 会持续尝试直到成功。

AI 应用普及还引入了新的攻击面。一家零售商的客服机器人升级为 LLM 版本后，系统立即暴露在 Prompt Injection 攻击之下。

**防守侧**：网络安全存在结构性不对称——一名攻击者可给数千万防御者制造海量工作负荷。传统防御软件本质上是被动的。传统人类红队有两个结构性缺陷：
1. 路径覆盖不完整：只选择阻力最小的路径完成目标。
2. 频率不足以应对环境变化：年度评估无法捕捉动态风险。

一次标准的人类咨询评估（两人干两周，加一周写报告）单次费用约 6 万美元。覆盖 1% 攻击面需约 100 万美元，100% 覆盖将高达 1 亿美元。

### Armadin 的产品

Armadin 的核心理念：先打造能执行国家级攻击行为的 agent，让它持续攻击自身系统，再用这些结果训练防御 agent，形成自主防御闭环。

产品形态是一个 cockpit 界面，支持 Web 应用渗透测试、infra 攻击、自定义应用攻击等场景。输出上避免第一代 AI 安全产品的高误报问题——根据漏洞的可利用性和业务影响两个维度排序，生成优先修复清单。

长期目标：五年后用 45 万个 AI agent 实时保护 3.2 万个实体，完全无需人工干预。

### 训练方法论

将攻击方法论结构化，转化为 agent 可直接学习的格式。团队先对每个 eval 环境进行人工评估，训练模型按专家真实思路执行攻击。

案例：在一次测试中，人类红队员通过观看长达一小时的 MP4 会议录像，发现白板上写着的服务账号和密码。这一过程被纳入 agent 训练后，agent 就具备了自动扫描长视频并识别敏感信息的能力。

高风险操作（远程代码执行、横向移动）需人类审批，低风险操作（主机发现、端口扫描等）完全自主执行。

### 商业飞轮

客户付费进行安全评估，同时为 Armadin 提供高质量的攻击方法训练数据。每个企业环境都有独特性，agent 在完成通用攻击路径后还需创造性地适应特定环境设计"创意解法"，这些才是真正有价值的训练素材。

算一笔经济账：原本需两周时间、花费约 6 万美元的人类渗透测试，agent 几分钟就能完成，token 成本不到一杯咖啡。企业将有能力实现全天候、100% 的攻击面覆盖。

### 三个真实案例

1. **暗网凭证泄露与未完成 2FA 账户接管**：agent 自主潜入暗网，提取 438 个被泄露的员工凭据，验证 7 个有效账户，其中一个因 2FA 未完成注册而被直接接管。
2. **软令牌密码失效**：攻破 RSA 服务器提取软令牌种子文件，发现统一导入密码，成功进入机密网络。
3. **航空舱单危机**：发现 Web 应用 SQL 注入漏洞，通过冷门端口 IBM MQ 消息队列协议（无内置身份验证），获得篡改航班舱单的读写权限。
