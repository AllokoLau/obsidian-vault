## 正文

### 郭达雅：从 DeepSeek 到字节

郭达雅，中山大学博士，2023 年加入 DeepSeek，深度参与代码智能与大模型推理方向研究。他是 DeepSeek V3、R1、Coder、Math 等一系列模型的核心贡献者，与团队共同提出的 GRPO（群体相对策略优化）算法在受限算力下显著降低训练资源需求，后被应用于 DeepSeek R1 训练。

郭达雅去年 10 月就有意离职，主要原因是他看好 agent 方向，但当时 DeepSeek 内部 agent 优先级不高。阿里、腾讯、字节均先后接触，阿里甚至给出了 post-train 负责人职位。最终因研究方向契合选择字节。

加入字节后职级 L8，薪资由现金、字节期权和豆包股三部分组成。年初豆包股价格上涨约 30%，保守估计四年后豆包股归属完毕，总收益可能高达数亿元，年均近亿元。此前 Seed 其他部门负责人在面试时曾以"要做世界第一梯队模型，集团支持很足"吸引人才。

### 字节的 AI Coding 战略

郭达雅的加入正值字节加大 AI 投入的关键节点。2026 年 1 月底，字节跳动 CEO 梁汝波在全员会上明确：2026 年重中之重是 AI 模型能力做到行业前列，通过助手整合已有产品服务。字节会继续加大对人才的投入，薪酬和激励在全球市场领先于头部水平。

Seed 内部正启动针对 agent 和 Coding 的组织整合。agent 的本质是代码理解、代码生成和程序综合，大模型代码能力直接决定了 agent 的能力上限。字节看重 AI agent 方向，下一代模型重点就是 agent 能力的优化。

值得注意的是，2025 年年底至今，中国大厂 AI 基模团队 Code 团队处于动荡期：阿里 Qwen Code 负责人惠彬原离职，如今 DeepSeek-Coder 第一作者郭达雅也选择离开。

### 海外 Coding 竞争格局

在海外，Coding 是 Anthropic 和 OpenAI 竞争最激烈的主战场，形成 Opus 4.6 vs GPT-5.4（模型层）和 Claude Code vs Codex（产品层）的双线对阵。

Claude Code 展现出惊人的吸金能力：2025 年 5 月正式上线，2026 年 2 月 ARR 即达 25 亿美元，不到一年超越 Cursor 两年多达到的 20 亿美元，增速超过 Salesforce、Slack 等软件企业早期。Coding 被证明是 AI 目前最有效的变现方式。

更重要的是，Claude Code 是 Anthropic 弯道超车的关键。2025 年底 Anthropic 全年收入 90 亿美元（OpenAI 214 亿），四个月后 ARR 暴涨至 300 亿美元，一举超过 OpenAI 的 250 亿 ARR。

### 中国市场的追赶

中国 AI 大模型公司在 Coding 方向上的布局早于大厂且更灵活：智谱 2025 年初确定 Coding 为重点方向，GLM-5 从发布后连续三次涨价，调用量仍增长 400%；月之暗面 K2.5 发布不到一个月，近 20 天累计收入超过 2025 年全年总额。

大厂方面，字节、阿里在 2025 年底才相继迭代自家 Coding 模型。接近团队的员工透露，"去年几乎每次训 Code 模型，上面都说是最后一次了"——当时大厂在该方向上的投入有限。2026 年初 OpenClaw 小龙虾爆火后，Coding 应用不再局限于编程群体，激发了广大普通用户的编程需求。

郭达雅的"天价"入职标志着中国大厂 AI Coding 人才争夺战进入新阶段，中国 Coding 和 agent 市场的竞争将愈加激烈。
