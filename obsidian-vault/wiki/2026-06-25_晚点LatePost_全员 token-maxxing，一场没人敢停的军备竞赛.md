## 正文

### 全员 token-maxxing

五源资本合伙人孟醒在 2026 年 3 月硅谷考察中发现，整个硅谷正陷入不计成本的 AI 投入狂潮。Meta 数万名工程师全员使用 Claude Code，内部甚至推出 token 消耗排行榜和"token legend"非官方头衔，同时 Meta 连续两轮裁员上万人——两条看似矛盾的动作实为同一件事的两面：先用 AI 提效再裁人。

代码安全是首先倒下的旗帜。Meta 最初自研内部 coding 工具 myclaw 但因"不好用"而放弃，最终放宽政策：只要不涉及客户数据，全公司爱用 Claude Code 就用。Google 禁止大多员工使用竞争对手工具，但 DeepMind 例外——Anthropic 为其做了私有化部署。AI-native 创业公司中，一个工程师年 token 预算已达 20 多万美元，接近一个工程师的薪资。

### 效率与营收的鸿沟

孟醒观察到：多家顶尖公司报告"百倍工程师""十倍效率提升"——以前 60 人一年的工作现在 2 人加 Claude Code 一周完成。但追问"营收增长了多少倍"时，无人给出正面回答。100 倍效率提升落到公司营收上只体现了 50% 或 1 倍。"用了这么多 token，公司应该基因突变成另外一种公司才对，但到底变成什么，我也不知道。"

### xAI 的人事地震

xAI 早期团队约 90% 已离职（导火索是 Tony Wu 离开后连锁反应），马斯克从 SpaceX 和特斯拉调人接管——"造火箭的人开始造模型了"。深层原因在于：制造业管理方式不适合大模型公司。马斯克擅长在长链条系统工程中抓关键杠杆点极限压缩时间线，但 xAI 缺少全局规划只有冲刺，"每个方向的负责人都在保自己的优先级"。

### Anthropic 的逆袭

Anthropic 是过去一年 AI 行业最戏剧性的逆转。2025 年底全年收入 90 亿美元（OpenAI 214 亿），差距超一倍。但四个月后 ARR 暴涨到 300 亿美元，翻三倍多，一举超过 OpenAI 的 250 亿 ARR。Coding 是胜负手——Claude Code 上线不到一年 ARR 达 25 亿美元，增速超过 Salesforce、Slack 等软件的早期阶段。

### 英伟达的权力中心

GPU 稀缺性重新加剧。API 稳定性变成稀缺资源——能稳定提供 API 服务的公司可卖官方价格 2-3 倍。英伟达通过资本和资源分配深度控制生态：同一轮融资出现"两个估值档位"的特殊结构，关系好的投资人进低估值档，英伟达和晚到者进高估值档。

上游内存厂商（Hynix、Samsung、Micron）产能扩建周期至少还需两年，2028 年前任何 AI 公司都无法靠堆算力显著拉开差距。

### 估值框架的崩塌

DCF 模型面临根本性挑战：只能预测 3 年而非 10 年，terminal value 的前提"稳定经营"在 AI 颠覆时代不成立。不在 AI 主航道的公司"像在等一颗核弹"——评估重点从"如果不被颠覆会怎样"变为"被颠覆时应对速度有多快"。SaaS 公司首当其冲被华尔街重新定价。

不同赛道的估值倍数也被打破：垂直 agent 约 5 倍 ARR、通用 agent 约 10 倍、模型公司 20-30 倍（Anthropic 26.7 倍）。

### 社会层面的震荡

Sam Altman 家遭燃烧弹袭击，FBI 搜出"你的最后警告"名单列有多名 AI 公司 CEO 姓名。全美约 100 个数据中心项目遭阻击，40 个将直接流产。缅因州通过法案全面禁止数据中心建设。旧金山爆发大规模反 AI 游行。孟醒总结硅谷的底色是"跟不上"——YC 跟不上、安全规则跟不上、管理跟不上、算力跟不上、估值框架跟不上、社会心理承受力跟不上。
