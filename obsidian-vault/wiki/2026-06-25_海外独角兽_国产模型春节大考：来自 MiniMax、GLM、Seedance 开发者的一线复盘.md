## 核心摘要

1. OpenRouter 过去一周平台调用量 Top 5 模型中有 4 个是国产开源（MiniMax M2.5、Kimi 2.5、GLM-5、DeepSeek V3.2），这 4 家的 token 总调用量占 Top 5 的 84.4%，占 Top 10 的 56.8%，国产模型在开发者社区影响力持续攀升。
2. Seedance 2.0 是春节模型的 magic moment，标志着国产大模型首次在全球范围内实现领先——在视频生成维度明显领先 Sora 2、Veo 3.1 等海外顶尖模型一个代际，效率提升 10-20 倍，从"抽卡逻辑"进化到"工业化可用"阶段。
3. MiniMax M2 选择 200B 参数 + 10B 激活的特殊模型尺寸，核心逻辑是平衡性能、速度和成本以适应 Agent 落地需求。Anthropic 最值得学习的经验是"Dogfooding"——全公司用 Agent Native 方式工作。
4. GLM-5（744B 参数，40B 激活）选择全面对标 Anthropic、押注工程师路线，引入 DSA 架构优化 long context coding 场景下的推理成本，定价约 3 美元（对标 Sonnet/Opus 的 15-25 美元），对开发者有极大吸引力。
5. Token 消耗已成为共识性新指标。2026 年 API 商业化的 Token 消耗预计实现 7-10 倍同比增长。国产模型面临低价竞争内卷、地缘政治导致的出海受限等挑战，ToC 产品出海可能是中国模型公司的最优路径。

## 正文

### 重要技术趋势

**模型能力跃迁从 Verifiable 向 Fuzzy 任务迁移**：大模型能力跃迁可基于"任务可被验证属性"和"解空间大小"二维象限理解。行业正将模型能力从可验证任务（数学、代码）推向模糊任务（深度研究报告、商业计划书）。未来 AI 需要能对开放任务做 self-criticizing、自我进化。

**Continual learning 是 2026 年的确定性趋势**：模型训练完后能否在部署阶段更新内部状态，让自己在特定任务上越做越好？长程任务的奖励信号非常稀疏，内在奖励机制（Intrinsic Reward）是可能的方法。

**Data 的重要性重新提权**：研发范式正在从 compute-bound 转移到 data-bound。算力边际重要性下降，真正拉开差距的点转为数据侧。中国在数据采集的劳动力组织和规模化上有优势。Seedance 2.0 之所以拉开差距，数据是核心变量，在长尾数据上做了大量精细化处理。

**Scaling Law 在国内依然能打**：GLM-5 把参数翻倍，Kimi 2.5 把数据集翻倍，都带来了显著智能收益。

**缺卡倒逼架构创新**：中国缺卡现状倒逼公司在架构创新上的积极性比硅谷更高，从底层解决推理效率问题。

### Seedance 2.0 的突破

字节跳动 Seedance 2.0 标志着国产大模型首次在全球范围内实现领先：
- **效率革命**：同等质量视频效率提升 10-20 倍，过去需一天的任务现在半小时内完成。
- **直出可用**：从"抽卡逻辑"进化到"工业化可用"，单次生成的 15 秒视频在语义理解和镜头衔接上已达商用门槛。
- **专业镜头语言**：精密语义理解、镜头切换，未来 6-12 个月视频创作门槛将彻底崩塌。
- **技术分层**：信息提取更分层化（纹理与材质微观提取），微动作/微表情建模更细，因果逻辑与物理关系理解更强。

字节在视频模型上的领先具有必然性：战略上 Seedance 是核心集团战略；组织上能做"三代同步"——一代做线上优化、一代做主力研发、一代做超前预研。

最难受的是做"上层工具/工作流"的团队——传统的链条式工作流正在被端到端模型吞没。

### MiniMax M2：定义问题的能力

MiniMax 认为做模型最重要的事情是"定义问题"。公开 Benchmark 只是对模型能力的一种观测，不代表真实市场价值。MiniMax 内部构建了 VIBE Bench 作为更全面的基准测试。

M2 选择 200B 参数 + 10B 激活的特殊尺寸，核心逻辑：Agent 需要全天候运行，用户可能同时并发 5-10 个 Agent，速度和成本与性能同等重要。从 Talkie 中得到的 learnings——这类产品使用密度极高，Claude 的价格放到全天候 Agent 上不可 affordable。

Anthropic 最值得学习的经验是 Dogfooding——全公司用 Agent Native 方式工作。MiniMax 已在全员推广，通过内部反馈收集 Benchmark 覆盖不到的边缘场景。

### GLM-5：工程师路线

智谱内部有过科学路线与工程师路线的辩论，最终选择押注工程师路线——追求更有经济价值的生产力任务。

Coding/Agent 任务与 Reasoning 任务有本质差异：Reasoning 在有限空间内做 heavy thinking，小模型靠强化学习就能刷高分；复杂的 code/agent 任务更依赖工程直觉。

GLM-5（744B 参数，40B 激活）的目标是在大多数场景下逼近 Claude Opus 效果。引入 DSA（DeepSeek Sparse Attention）架构优化 long context coding 场景下的计算成本——处理如 200K 长度序列时，只关注局部窗口内经过筛选的有限 token，Prefilling 阶段计算量显著降低。

定价约 3 美元/百万 Token，对标 Sonnet/Opus 的 15-25 美元，对开发者有极大吸引力。国内开源模型对 Claude skills 生态兼容度很好，切换相对顺滑。

### Token 消耗与商业化

Token 消耗已成为共识性新指标。火山云追踪显示：2024 年主战场在 research 类场景；2024-2025 上半年三足鼎立（互动娱乐、客服/销售、教育各占约 15%）；2025 年中视频模型发力；2025 年下半年至今 Agentic 场景占比达 35%，预计 2026 年占到一半。

2026 年 API 商业化 Token 消耗预计实现 7-10 倍同比增长。视频生成模型对 Token 消耗有巨大杠杆效应——单次 query 消耗远超文本。

国产模型面临低价竞争内卷风险：GLM 定价约 3 美元 vs Anthropic 的 15-25 美元。可能出现类似光伏、锂电的局面——全世界在用但利润微薄。

### 出海策略

地缘政治限制国产模型直接向美国企业提供 API。创新出海路径包括：
1. 借壳"OpenRouter 生态"出海——Fal.ai 估值已达 45 亿美元，平台上核心模型很多是中国的。
2. "延迟开源"策略——每代模型训练完后先设 3-6 个月 Preview 期，仅授权海外合作伙伴，下一代发布时再开源上一代。

长期看，中国厂商的优势在于产品出海而非直接卖 API。历史上中国真正做成的全球化产品全是 ToC 的（TikTok、Temu、Shein）。Seedance 2.0 让人看到了"再造几个大 C 端平台"的机会。
